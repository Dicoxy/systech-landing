## Стек
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React

## Главные правила

### Код
- Компоненты до 100 строк
- Один компонент = один файл
- Mobile-first
- Семантический HTML

### Дизайн
- Фон: #0a0a0f
- Акцент: #00ff88
- Шрифты: Space Grotesk (заголовки), Inter (текст)

### Процесс
- Всегда включай plan-mode
- Одна задача за раз
- После каждой задачи — проверка на localhost
- Не выдумывай, бери из shadcn
